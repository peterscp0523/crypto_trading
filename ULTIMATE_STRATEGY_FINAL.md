## 🔧 전략의 핵심 원리

### 1. 다층 방어 시스템

```python
# 시장 상황에 따라 자동 전환
강세장 → 모든 Layer 활성화
약세장 → Buy & Hold만 유지
횡보장 → Volatility만 활성화
```

### 2. 리스크 분산

```
각 Layer는 독립적으로 작동
한 전략 실패해도 다른 전략이 보완
포트폴리오 효과로 MDD 최소화
```

### 3. 검증된 지표 조합

**모멘텀 스코어 (100점 만점)**:
- 다중 이동평균 배열 (40점)
- 가격 모멘텀 다중 기간 (30점)
- 볼륨 트렌드 (15점)
- RSI 모멘텀 (15점)

**진입 기준**:
- 강한 모멘텀: 80점+
- 중간 모멘텀: 60-80점
- 약한 모멘텀: 진입 안 함

### 4. 엄격한 리스크 관리

```python
# 각 전략마다 손절 설정
Momentum Trend: 20일 저점 -2%
Momentum Swing: 10일 저점 -3%
Volatility: -1.5 ATR

# 트레일링 손절
수익 5% 이상 → 손절가를 진입가 +2%로 상향
→ 최소 2% 수익 확보
```
